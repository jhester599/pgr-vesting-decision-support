# Root Archive

This root `archive/` directory stores retired code and tests that are no
longer part of the active runtime or pytest collection.

Use this directory for archived executable artifacts only:

- retired one-off research scripts
- companion tests for those retired scripts

Use `docs/archive/` for historical documents, external reports, peer reviews,
and old planning material.

Do not add new production code here. If archived code becomes useful again,
promote the reusable logic into `src/` with tests.

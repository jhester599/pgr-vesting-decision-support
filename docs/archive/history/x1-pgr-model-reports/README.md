# x1 PGR Model Research Reports

This directory archives the two external deep-research reports used as source
context for the x-series absolute PGR forecasting and special-dividend research
lane.

Archived reports:

- `20260421-pgrmodels-chatgpt.md`
- `20260421-pgrmodels-gemini.md`

Both reports support starting with multi-horizon absolute direction
classification, benchmarking direct forward-return regression, and treating the
BVPS x P/B path as a structural valuation benchmark. The Gemini report is
especially explicit about overlapping-target leakage risk and hurdle-style
classification. The ChatGPT report is broader on feature inventory and
repo-aligned implementation sequencing.

These reports are informational source material only. They do not alter the
main `v###` roadmap, production monthly workflow, or any promoted model
behavior.
